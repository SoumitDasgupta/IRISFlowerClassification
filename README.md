# 🌸 Iris Flower Classification using Logistic Regression and Random Forest

## 📌 Overview

This project was developed as part of my **4-Week Summer Internship Programme in Data Science 2026** at the **Institute of Data Engineering, Analytics and Science Foundation (IDEAS-TIH), Indian Statistical Institute (ISI), Kolkata**.

The objective of this project is to classify Iris flowers into three different species using supervised Machine Learning algorithms. Two classification models—**Logistic Regression** and **Random Forest Classifier**—were implemented, trained, and evaluated to compare their performance.

---

## 📊 Dataset

The project uses the famous **Iris Dataset**, introduced by Ronald Fisher.

### Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

### Target Classes
- Setosa
- Versicolor
- Virginica

Dataset Source:
https://archive.ics.uci.edu/ml/datasets/iris
